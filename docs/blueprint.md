# **App Name**: Simpler Starter

## Core Features:

- Basic Display: Display the 'Hello, World!' message on the page.

## Style Guidelines:

- Primary color: Light blue (#A7D1E7) to provide a calm and simple background.
- Background color: Very light blue (#E8F4FC), near-white to give a minimalistic feeling.
- Accent color: Darker blue (#5D9EBC) to provide enough contrast with text.
- Font: 'Inter' sans-serif, to maintain a clean and readable appearance.
- Center the 'Hello, World!' text on the screen for a simple, clean presentation.
